**Problema:** El taller manejaba de forma manual la gestión de servicios (pintura, latonería, polichada, recubrimiento cerámico), inventario y cotizaciones, lo que generaba demoras y falta de control.

**Solución:** Aplicación web completa con panel administrativo para gestionar inventario, servicios y cotizaciones. Sitio informativo con catálogo de servicios, galería de trabajos realizados y canales de contacto. Diseño responsivo con enfoque en experiencia de usuario premium.

**Stack:** HTML5, CSS3, JavaScript, Node.js, base de datos SQL, despliegue en Railway.
